# 🤝 Contributing to MCP Servers Hub

Thank you for helping us build the best MCP servers directory!

---

## 📋 Requirements

### ✅ Acceptance Criteria

| # | Rule | Required |
|---|---|---|
| 1 | Must be a valid MCP (Model Context Protocol) server | ✅ Yes |
| 2 | Must have a public GitHub repository | ✅ Yes |
| 3 | README / documentation must exist | ✅ Yes |
| 4 | Must be actively maintained (commits within last 6 months) | ✅ Yes |
| 5 | Description under 80 characters | ✅ Yes |
| 6 | Placed in the correct category | ✅ Yes |
| 7 | Alphabetical order within category | ✅ Yes |

### ❌ Auto-Reject Rules

- ❌ Multiple servers in one PR (1 server per PR)
- ❌ Dead / archived repositories
- ❌ No documentation or README
- ❌ Not an actual MCP server
- ❌ Spam or promotional content
- ❌ Broken GitHub links

---

## 🚀 How to Add an MCP Server

### Step 1: Fork this Repository

### Step 2: Edit README.md
Add your MCP server in the correct category using this format:

```markdown
| Server Name | Short description (under 80 chars) | ⭐ Star count | [GitHub](link) |
```

### Step 3: Submit a Pull Request
- Title format: `Add: [Server Name] MCP`
- Fill out the PR template
- Wait for review (usually within 48 hours)
